# CLI 슈팅 게임 🚀

간단하지만 재미있는 터미널 기반 우주 슈팅 게임입니다!

## 게임 방법

### 조작법
- **A**: 왼쪽으로 이동
- **D**: 오른쪽으로 이동
- **SPACE**: 총알 발사
- **Q**: 게임 종료

### 게임 규칙
- 위에서 내려오는 적(▼)을 총알(|)로 격파하세요
- 적을 격파하면 10점을 얻습니다
- 적이 바닥에 도달하면 게임 오버!
- 최대한 많은 점수를 획득하세요!

## 실행 방법

### Windows
```bash
python shooting_game.py
```

### Linux/Mac
```bash
python3 shooting_game.py
```

또는 실행 권한 부여 후:
```bash
chmod +x shooting_game.py
./shooting_game.py
```

## 요구사항
- Python 3.6 이상
- 추가 패키지 설치 불필요 (표준 라이브러리만 사용)
